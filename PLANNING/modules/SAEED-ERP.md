## 🔗 التكامل مع BI ERP

### ما يستقبله من BI ERP:
```
✅ قائمة المنتجات (بدون سعر الشراء)
✅ أسعار البيع
✅ كميات المخزون
✅ حالة المنتجات
```

### ما يُرسله لـ BI ERP:
```
→ الفواتير المُنشأة
→ المبيعات المكتملة
→ طلبات الضمان
→ المرتجعات من الزبائن
```

---

## ⚠️ ملاحظات مهمة للتكامل

1. **لا يرى سعر الشراء أبداً**
2. **لا يرى هامش الربح**
3. **لا يمكنه تعديل أسعار البيع**
4. **الفواتير تُرسل تلقائياً لـ BI ERP**
