# مصدر بيانات سكربتات التحليل المالية

---

## مجلد التصدير

مصدر البيانات لسكربتات التحليل المالية (مثل [scripts/analyze-financials.py](../scripts/analyze-financials.py)) هو مجلد **تصدير قراءة فقط**:

- **المسار:** `data/morabaa-export/`
- **المحتوى:** ملفات JSON مُصدَّرة من نظام آخر (Morabaa أو مصدر بيانات خارجي).

السكربتات تقرأ من هذه الملفات ولا تعدّلها.

---

## تحذير

- **لا تُستخدم مخرجات هذه السكربتات كأساس وحيد** لقرارات مالية أو تدقيق رسمي دون التطابق مع البيانات في نظام BI Management الأساسي عند الحاجة.
- التصدير قد لا يكون محدثاً في الوقت الفعلي؛ للتأكد من دقة الأرقام يُفضّل الرجوع إلى قاعدة البيانات الرسمية أو تقارير الـ Backend.

---

## التوصية لأغراض التدقيق

- عند الحاجة لتدقيق أو تقارير رسمية: إنتاج تقارير مماثلة من **قاعدة البيانات الرسمية** عبر:
  - Backend API (مثلاً مسارات التقارير `/api/reports`)، أو
  - تقارير مدمجة في النظام عند توفرها.

هذا يضمن أن مصدر البيانات معتمد ومتسق مع سجل التدقيق والعمليات المسجلة في النظام.
