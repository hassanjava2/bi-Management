## 🔗 التكامل مع BI ERP

### ما يستقبله من BI ERP:
```
✅ قائمة المنتجات المتاحة
✅ الصور والمواصفات
✅ أسعار البيع
✅ كميات المخزون (متاح/نفذ)
✅ التصنيفات
```

### ما يُرسله لـ BI ERP:
```
→ الطلبات الجديدة
→ معلومات الزبائن
→ طلبات الصيانة/الضمان
→ إحصائيات الزوار
```

---

## ⚠️ ملاحظات مهمة للتكامل

1. **لا يرى سعر الشراء أبداً**
2. **المنتجات تُحدّث تلقائياً من BI ERP**
3. **الطلبات تُرسل فوراً لـ BI ERP**
4. **المخزون يُحدّث بالوقت الفعلي**
